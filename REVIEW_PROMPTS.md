# Промпты для независимой проверки архитектурного анализа

**Цель:** Организовать перекрёстную проверку `ARCHITECTURE_REVIEW.md` несколькими независимыми AI-агентами.
**Метод:** Каждый агент получает общий контекст + специализированную область для глубокого анализа.
